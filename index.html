How to Root Redmi 10a?

First We need to download all the tools listed below to prepare for rooting

Prequesties:

***Mobile Must have Sim card***

  1. MI OEM unlock Tool
  2. XiaoMitool V2
  3. Redmi 10a fastboot rom
  4. Magisk App
  5. Root Checker App
  6. android platform-tools

Rooting Method

* Modifying redmi 10a fastboot rom boot.img by magisk app

* Booting the redmi 10a with modified boot.img using fastboot tool in platform-tools

Step 1: Enable Developer options

Turn on developer option by following steps

  1. Go to Settings
  2. Go into About Phone
  3. Click All Specs
  4. Tap MIUI version until you see , you are a developer
  5. Now back, Developer Options is shown in the Additional Settings
  6. Enter into Developer options, Turn on
  7. Enable usb debugging

Step 2: Unlocking Bootloader

  1. Enable OEM unlocking
  2. Click Mi Unlock Status
  3. Click Add an Account
  4. Add your Mi account
  5. Download mi oem unlock tool using above link
  6. Connecting Your Mobile to Pc using Usb cable
  7. After Installing, Login with Your Mi Account Or using Qr code Scanner in accounts Settings
  8. After Logged in,You See Your Devices If not install XiaoMitool V2-> Select I want to Mod-> Installing the driver
    after that back to mi unlock tool now your device get recognized
  9. Click unlock after that, It Says 168 hours to unlock
  10. Wait for one week
  11. One Week Later, Again run the mi unlock tool to unlock. Now your device get unlocked after reboot.
  12. Go to settings ->Additional settings -> Developer option -> Mi unlock Status. Now you see This device is unlocked.

Now We are Sucessfully unlocking the bootloder.

Step 3: Installing platform-tools and magisk

  1. Download and Install the platform-tools in your pc using above link
  2. Installing Magisk app in your mobile from above link.

Step 4: Modifying boot.img

  1. Download redmi 10a fastboot rom from above link, In my case my redmi 10 a mobile is global version and miui version 12.5.16.0
    You can download your model rom by search on miuirom.com
  2. unzip it
  3. Inside the folder -> Images -> copy the boot.img and vbmeta.img to the platform-tools folder
  4. Go to platform-tools folder
  5. We need to modify the boot.img for rooting so,pushing the boot.img to your mobile using adb by following commands
  6. adb devices

    7. wait for your device shown in list of devices attached

  7. adb.exe push boot.img /sdcard/
  8. Now see your mobile has recived
  9. Now, open magisk app -> Install -> Select and Patch a file
  10. Select the boot.img we pushed to mobile
  11. It returns the magisk patched boot.img
  12. getting the magisk patched boot.img using adb pull by following command
  13. adb pull magisk_patched.img

Step 5: Flashing magisk_patched.img

Flashing by following commands

  1. adb reboot bootloader
  2. fastboot flash boot magisk_patched.img
  3. fastboot flash --disable-force-verification --disable-verity boot vbmeta vbmeta.img
  4. fastboot reboot

Step 6: Root Check

Installing the root Checker app in mobile using above link

Click verify root -> Grant Permission forever when it is popped up a window.

after that, you can the see text, Congratulations! Root access is properly installed on this device!

Enjoy! Your Redmi 10a Phone is now get rooted.
